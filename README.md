# CodeAlpha_AIChatbot

## Project Overview

AI Chatbot is a Java-based conversational application that simulates human interaction using predefined rules and responses. The chatbot can answer common questions and provide interactive communication through a console interface.

## Features

* Interactive conversation
* Rule-based response system
* Frequently asked questions support
* Exit command functionality
* Real-time user interaction

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Scanner Class
* Conditional Statements

## Project Structure

* ChatBot.java

## How to Run

1. Open the project in VS Code or any Java IDE.
2. Compile the program:
   javac ChatBot.java
3. Run the application:
   java ChatBot
